## Requirements

This guide provides step-by-step instructions for manually installing SylCore WoTLK on a Linux system.
While we offer an automatic installation script, this manual guide is for users who prefer to understand
and control each step of the installation process

:::caution[Tools Needed]
1. Ubuntu 24.04 (tested and recommended)
2. Basic knowledge of Linux command-line operations
3. Administrator (sudo) privileges
4. Stable internet connection
:::
---

### Dependencies

First, we need to install all required packages for SylCore:

```bash
# Update package lists
sudo apt-get update

# Install required dependencies
sudo apt-get install -y git cmake make gcc g++ clang libmysqlclient-dev \
libssl-dev libbz2-dev libreadline-dev libncurses-dev mysql-server \
libboost-all-dev unzip
```
