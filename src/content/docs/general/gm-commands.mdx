---
title: GM Commands
description: A structured list of GM commands with their syntax, security levels, and descriptions.
---

GM commands can be entered mainly in two ways: directly into the world console window or in-game via the chat console. In the world console, the leading dot (.) is optional, but in-game, all commands must start with a leading dot.

> **Note:** Some commands require selecting a player or creature. These commands cannot be used in the world console.

> **Security Levels:**
> - 0: Player
> - 1: Moderator
> - 2: Game Master
> - 3: Administrator
> - 4: Console
